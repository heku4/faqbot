### FAQBot

*FAQBot* quickly answers questions. 
Questions and answers to them are stored in the file `faq.json`.
The application uses the [Telegram-bot](https://github.com/TelegramBots) client.

#### faq.json scheme
```
[
  {
    "Question": "When?",
    "Answer": "Now!"
  },
  {
    "Question": "Where?",
    "Answer": "There!"
  }
 ]
```
